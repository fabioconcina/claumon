claumon

Real-time Claude Code dashboard. Single binary, zero config, one browser tab.

Rate limit gauges Token costs Session history Memory browser Consolidation suggestions Live updates
Download GitHub
$ go install github.com/fabioconcina/claumon@latest
$ claumon --open

What you get

◕

Rate limit gauges

Live 5-hour and 7-day utilization with reset countdowns. Color-coded so you know when to slow down.

▦

Token breakdown

Input, output, cache read, cache write tokens per session. Estimated API-equivalent costs at current pricing.

◆

Session history

Browse active sessions with project, model, token counts, and costs. Click into any session to see the full conversation.

⚙

Memory browser

Search across all CLAUDE.md files, rules, and auto-memories. Filter by type, spot stale entries, get consolidation suggestions for overlapping files.

❖

Memory graph

Interactive visualization of memory relationships. See cross-project connections, filter by project, click to navigate.

⚡

Real-time updates

Server-Sent Events push changes instantly. No polling, no manual refresh. Gauges, sessions, and memories update live.

Three steps

1

Install

Download a prebuilt binary or go install. One file, no dependencies.

2

Run

Launch claumon --open. It picks up your Claude credentials automatically.

3

Use Claude Code

Keep the tab open. Everything updates in real time as you work. Historical data persists in a local SQLite database.
